# Демо приложение "TASK LIST"

- Добавление задачи;
- Удаление задачи; 
- Изменение задачи;
- Возможность сделать задачу выполненной или невыполненной;
- Очистка списка задач;
- Хранение данных - localStorage.

## Зависимости

- [Create React App](https://github.com/facebookincubator/create-react-app)
- [Redux](http://redux.js.org/)
- [Underscore.js](http://underscorejs.org/)
- [Semantic UI React](https://react.semantic-ui.com/introduction/)

## Требования

- [Node.js](https://nodejs.org)

## Установка

1. git clone git@github.com:Kamyhin/todo-list.git
2. npm install

## Команды

### `yarn start`

Запускает приложение в режиме разработки.
Автоматически открывает приложение в браузере по адресу [http://localhost:3000](http://localhost:3000).

### `yarn build`

Компилирует приложение в режиме production  в папку `build`.

### `yarn eject`

Переностит конфигурационные файлы в дирректорию проекта, устанавливает все необходимые зависимости.
